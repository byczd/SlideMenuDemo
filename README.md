### 实现原理   
本Demo几乎以最简化的方式来实现一种侧边栏效果,方便理解其原理,所以,只有tap手势过渡效果,并未实现pan的侧滑效果.  
  
实现这种效果,要用容器控制器的概念.通过容器控制器中不同容器视图的切换,来实现.  
  
实现的难点在于容器视图切换中,动画效果平缓自然.
